Constructor & Destructor Documentation

ForLeaseEngine::Entity::Entity ( )

Constructor for a new Entity. Sets the ID of this Entity to a new, unique ID that is derived from the total number of entities created before it plus one. It also initializes the bitfield mask of components added to this entity to None.

ForLeaseEngine::Entity::~Entity ( )

Destructor for an Entity. Destroys all components attached to it.
